# 전처리 근거 및 과정

본 문서는 제10차 산업안전보건 실태조사(2021) 원자료로부터 최종 분석 데이터셋(n=1,375, 17변수)을 생성한 과정을 기록한다.

---

## 1. 연구 자료

한국산업안전보건공단 산업안전보건연구원이 실시한 **제10차 산업안전보건 실태조사(2021)** 원자료. 공사금액 50억 원 이상 건설 사업장 **1,502개**를 대상으로 하였다.

---

## 2. KEY PAPER와의 전처리 방식 비교

| 항목 | KEY PAPER (의료) | 본 연구 (건설안전) |
|---|---|---|
| 데이터 출처 | 합성(Synthetic) | **실제 공식조사** |
| 초기/최종 표본 | 1,500 → 1,200 | 1,502 → 1,375 |
| 결측 처리 | KNN imputation (k=5) | **Listwise Deletion** |
| 제외 사유 | 의학적 (말기암 등) | 설문 무응답 (Skip Logic) |

**왜 KNN 대신 Listwise?**: 본 데이터의 결측은 설문 응답 거부/조건부 미응답이므로, 무응답 자체가 사업장의 특성을 반영한다. 인위적 대체는 편향을 유발하므로 listwise가 적절. 박천수(2023, 2024, 2025)의 동일 데이터 선행연구도 동일 방식 채택.

## 5. 특징 공학

**Label Encoding**: 공사규모(소1·중2·대3) 등 범주 번호 그대로 투입.

**Z-score 표준화**: 모든 연속형 변수를 평균 0, 표준편차 1로 변환.

$$z = \frac{x - \mu}{\sigma}$$
